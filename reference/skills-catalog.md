# Skills Catalog

All agent skills organized by category, with primary role assignments.

## Essential Skills (all teams)

### Mermaid

Generate diagrams from structured data. Supported diagram types:

- `flowchart` — process flows, decision trees
- `sequenceDiagram` — interaction sequences between components or actors
- `erDiagram` — entity-relationship models for data architecture
- `gantt` — project timelines and milestone scheduling
- `stateDiagram` — state machines for lifecycle documentation
- `architecture` — system topology and deployment views
- `pie` / `quadrantChart` — proportional breakdowns and comparative positioning

### Table Generator

Produce structured comparison and specification tables:

- Comparison tables (vendor vs. vendor, option vs. option)
- Spec tables (product specifications, feature matrices)
- Pricing tables (unit cost, quantity, subtotal, TCO summary)

### FP / Cost Estimation

Calculate function points and derive development cost and TCO figures.

- Function point calculation (IFPUG or simplified)
- Development cost estimation from FP totals
- TCO projection (3-year, 5-year horizons)
- Primary owners: BA, TA

### RFP Traceability Matrix

Map every RFP requirement to the proposal section(s) that address it.

- Requirement ID -> SSOT section -> response status
- Gap identification for unanswered requirements
- Primary owners: BA, Overseer

### Document Parser

Parse uploaded files into SSOT-compatible structured data.

- Supported formats: PDF, DOCX, PPTX
- Extracts headings, tables, lists, and inline metadata
- Primary owners: all teams

## Quality Differentiation Skills

### Glossary

Maintain a unified terminology dictionary across the proposal.

- Add / update / delete terms
- Flag inconsistent usage across SSOTs
- Primary owners: Overseer, Critic

### Regulatory Checklist

Verify compliance with financial security guidelines and industry regulations.

- Clause-by-clause checklist generation
- Gap analysis against proposal content
- Primary owners: Overseer, Critic

### Reference Manager

Store and retrieve delivery case studies in a structured format.

- Fields: client, project name, period, outcome, current status
- Reusable across proposals
- Primary owner: Researcher

### Competitive Analysis

Build advantage-comparison tables for competitive positioning.

- Feature-by-feature competitor mapping
- Strength / weakness summary
- Primary owner: SA

## Output Skills

### MD -> PPT / PDF / HTML

Render confirmed SSOTs into presentation-ready formats.

- PPT: slide deck with branded template
- PDF: print-ready document
- HTML: web-viewable output

### Version Diff

Compare two versions of the same SSOT side by side.

- Field-level diff highlighting
- Change summary with severity tags

### Executive Summary

Generate a 1-page summary from long SSOTs.

- Key metrics, decisions, and recommendations extracted
- Suitable for executive briefing decks
